# 图片编辑与AI生成应用

一个基于Flask的图片编辑与AI生成Web应用，支持图片上传、鼠标框选、API调用和结果展示。

## 功能特点

### 📁 图片上传与框选
- 支持JPG、PNG、JPEG格式图片上传
- 鼠标左键拖拽框选
- 6种颜色选择（蓝、绿、橙、灰、白、红）
- 3种粗细可选（细3px、中6px、粗10px）
- 清除框选功能

### ⚙️ API配置
- 支持OpenAI格式的API
- 可配置API Key、API URL和Model
- 默认配置已预设
- API Key可显示/隐藏

### 🎯 功能按钮管理
- 添加新功能按钮，自定义名称和提示词
- 删除不需要的按钮
- 折叠式面板设计

### 🚀 生成与展示
- 点击功能按钮发送框选图片和提示词到API
- 支持多次生成，结果依次展示
- 每个结果独立加载状态
- Markdown图片链接自动解析
- 图片防盗链处理
- 下载生成的图片

### 💾 结果管理
- 多个结果卡片式展示
- 每个结果可单独下载
- 一键清除所有结果

## 安装与运行

### 方式一：Docker部署（推荐）

#### 拉取镜像
```bash
docker pull zhoushu1/ai-image-selection-and-modification
```

#### 运行容器
```bash
docker run -d -p 5000:5000 zhoushu1/ai-image-selection-and-modification
```


### 方式二：直接运行

#### 依赖安装
```bash
pip install flask requests pillow
```

#### 启动应用
```bash
python app_complete.py
```

### 访问应用
打开浏览器访问：http://localhost:5000

## 使用说明

### 1. 上传图片
- 点击"选择文件"上传图片
- 图片会在第一列显示

### 2. 框选（可选）
- 在图片上按住鼠标左键拖拽进行框选
- 可选择框的颜色和粗细
- 点击"清除框选"重置

### 3. 配置API
- 展开"API 配置"面板
- 确认API Key、URL和Model正确
- 默认配置已预设，可直接使用

### 4. 添加功能按钮
- 展开"添加新功能按钮"面板
- 输入按钮名字和提示词
- 点击"保存按钮"

### 5. 生成结果
- 在"选择功能生成"区域点击按钮
- 结果会在第二列显示
- 可多次点击生成多个结果
- 每个结果都有独立的加载状态

### 6. 下载图片
- 点击结果卡片中的"📥 下载结果"按钮
- 图片会自动下载
