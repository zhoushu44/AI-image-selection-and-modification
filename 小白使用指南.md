## 🎒 准备工作

在开始之前，确保你有：

1. **一台电脑**（Windows/Mac/Linux 都可以）
2. **网络连接**（需要访问 Supabase 和 API）
3. **邮箱地址**（用于注册 Supabase 和应用账号）
4. **Python 3.7 或更高版本**（如果没有，先去 https://python.org 下载安装）

---

## 🔑 第 1 步：注册 Supabase 账号

Supabase 是一个免费的数据库服务，我们用它来存储用户数据和配置。

### 操作步骤：

1. 打开浏览器，访问：https://supabase.com
2. 点击右上角的 **"Start your project"** 按钮
3. 使用你的邮箱注册账号（或者用 GitHub 账号快速登录）
4. 验证邮箱（如果需要）

---

## 🏗️ 第 2 步：创建 Supabase 项目

### 操作步骤：

1. 登录后，点击 **"New Project"** 按钮
2. 填写项目信息：
   - **Name**：给项目起个名字，比如 `AI图片编辑`
   - **Database Password**：设置一个数据库密码（**一定要记住！**）
   - **Region**：选择离你最近的地区（中国用户建议选 `Singapore`）
3. 点击 **"Create new project"**
4. 等待 1-2 分钟，项目创建完成！

---

## 🔐 第 3 步：获取密钥信息

项目创建好后，我们需要获取 3 个重要信息。

### 操作步骤：

1. 在项目页面，点击左侧菜单的 **"Settings"**（齿轮图标）
2. 点击 **"API"**
3. 在这里你会看到：
   - **URL**：这是你的项目地址，格式类似 `https://xxxxx.supabase.co`
   - **anon public**：这是公开密钥（Anon Key）
   - **service_role secret**：这是秘密密钥（Secret Key）

4. **重要！** 把这三个信息复制到一个记事本里，稍后会用到！

### 配置 Supabase 认证设置：

1. 点击左侧菜单的 **"Authentication"**
2. 点击 **"Providers"**
3. 找到 **"Email"**，确保它是开启状态
4. （可选）如果不想验证邮箱，把 **"Confirm email"** 开关关掉

---

## ⚙️ 第 4 步：配置应用

现在我们需要把刚才获取的密钥配置到应用里。

### 操作步骤：

1. 找到项目文件夹里的 `app_complete.py` 文件
2. 右键点击，选择用记事本或代码编辑器打开
3. 找到这几行代码：

```python
SUPABASE_URL = "https://your-project.supabase.co"
SUPABASE_ANON_KEY = "your-anon-key"
SUPABASE_SECRET_KEY = "your-secret-key"
```

4. 把这三个值替换成你刚才从 Supabase 复制的信息：
   - 把第一个引号里的内容换成你的 **URL**
   - 把第二个引号里的内容换成你的 **anon public**
   - 把第三个引号里的内容换成你的 **service_role secret**

5. 保存文件！

---

## 📊 第 5 步：创建数据库表

现在我们需要在 Supabase 里创建数据库表。

### 操作步骤：

1. 回到 Supabase 项目页面
2. 点击左侧菜单的 **"SQL Editor"**
3. 点击 **"New query"**
4. 打开项目文件夹里的 `sql` 文件夹
5. 打开 `create_tables_simple.sql` 文件
6. **全选**文件里的所有内容（Ctrl+A 或 Cmd+A）
7. **复制**（Ctrl+C 或 Cmd+C）
8. 回到 Supabase 的 SQL Editor，**粘贴**到输入框里（Ctrl+V 或 Cmd+V）
9. 点击 **"Run"** 按钮执行
10. 看到 "Success" 提示就表示成功了！

---

## 📦 第 6 步：安装 Python 依赖

现在我们需要安装运行这个应用所需的软件包。

### 操作步骤：

1. 打开 **命令提示符**（Windows）或 **终端**（Mac/Linux）
   - Windows：按 Win+R，输入 `cmd`，回车
   - Mac：按 Cmd+空格，输入 `terminal`，回车

2. 进入项目文件夹：
   ```bash
   cd 你的项目文件夹路径
   ```
   （提示：可以在文件管理器里复制路径，然后粘贴到命令行里）

3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

4. 等待安装完成，看到 "Successfully installed" 就表示成功了！

---

## 🚀 第 7 步：运行应用

激动人心的时刻到了！让我们启动应用！

### 操作步骤：

1. 在命令提示符或终端里，确保还在项目文件夹
2. 运行这个命令：
   ```bash
   python app_complete.py
   ```

3. 看到类似这样的提示就表示成功了：
   ```
    * Running on http://127.0.0.1:5000
   ```

4. **不要关闭这个窗口！** 让它一直运行着

---

## 🎮 第 8 步：使用应用

应用已经启动了！现在让我们来使用它。

### 操作步骤：

1. 打开浏览器，访问：http://localhost:5000
2. 你会看到登录页面

#### 📝 注册账号：

1. 点击 **"注册"** 标签
2. 输入你的邮箱（可以是任意邮箱）
3. 设置密码（至少 6 位）
4. 点击 **"注册"** 按钮
5. 注册成功后会自动登录

#### 🎁 新用户福利：

- 注册成功自动获得 **10 积分**
- 自动获得 **8 个默认功能按钮**

#### 📤 上传图片：

1. 点击 **"选择文件"** 按钮
2. 选择你想编辑的图片（支持 JPG、PNG）
3. 图片会显示在左侧

#### ✏️ 框选（可选）：

1. 在图片上按住鼠标左键拖拽
2. 可以选择框的颜色和粗细
3. 点击 **"清除框选"** 可以重置

#### 🎨 生成图片：

1. 在右侧点击任意功能按钮
2. 等待几秒钟，结果会显示出来
3. 每次生成扣除 **2 积分**
4. 可以多次点击生成多个结果

#### 💎 领取每日积分：

1. 点击顶部的 **"积分"** 按钮
2. 点击 **"领取每日积分"**
3. 每天可以免费领取 **10 积分**

#### 🔧 添加自定义按钮：

1. 展开 **"添加新功能"** 面板
2. 输入按钮名字（比如"卡通化"）
3. 输入提示词（比如"把这张图片变成卡通风格"）
4. 点击 **"保存按钮"**

#### 💾 下载图片：

1. 点击结果卡片里的 **"下载结果"** 按钮
2. 图片会自动保存到你的电脑

## ❓ 常见问题解答

### Q: 提示 "Python 不是内部或外部命令" 怎么办？

A: 说明 Python 没有正确安装或者没有添加到环境变量。
- 重新安装 Python，安装时勾选 "Add Python to PATH"
- 或者参考网上的"Python 环境变量配置"教程

### Q: pip install 很慢或者失败怎么办？

A: 可以使用国内镜像源：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: 注册失败提示 "Database error saving new user"？

A: 检查以下几点：
1. 确认已经在 Supabase 执行了 `create_tables_simple.sql`
2. 确认 `app_complete.py` 里的密钥配置正确
3. 确认 Supabase 项目状态正常

### Q: 应用能打开，但没有按钮显示？

A: 刷新页面试试，新用户首次登录会自动初始化按钮。

### Q: 生成图片一直没反应？

A: 检查：
1. 网络连接是否正常
2. 积分是否足够（至少需要 2 积分）
3. API 配置是否正确（需要在 Supabase 的 api_config 表里配置）

### Q: 怎么停止应用？

A: 在运行应用的命令窗口里按 **Ctrl + C** 就可以停止。

### Q: 下次怎么重新启动应用？

A:
1. 打开命令提示符/终端
2. 进入项目文件夹
3. 运行 `python app_complete.py`
